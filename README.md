# 個人網頁製作紀錄

## 介紹

&emsp;&emsp;這是我第一次嘗試製作網頁，因為我才大一，經歷不多，所以我盡可能的放讓許多讓大家可以認識我的東西，包括高中以前做的許多小作品。雖然有模板可以修改，但面對龐大的原始碼還是會有點驚慌失措，但熟悉了以後發現其實並不難。

## 修改/增加部分

&emsp;&emsp;這個模板看似漂亮，但實則缺失許多功能（或許翉來就不該有），所以我作了一些修改，如下：

1. 點選按鈕自動下滑。
2. 照片瀏覽器功能。
3. 文字加陰影
